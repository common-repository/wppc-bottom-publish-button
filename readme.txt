== Description ==

Save a lot of your time by scrolling less in WP admin! Show buttons (Scroll to top and Publish/Update and Save Draft and Preview) in the bottom of the screen when user scrolls near bottom.

Simple jQuery script of this plugin duplicates Update / Publish / Save Changes / Save Draft / Preview Changes button for posts, pages, custom posts from bottom of screen, so that it stays on site while you are scrolling your admin page. 

== Installation ==

1. Upload the entire 'wppc-bottom-publish-button' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
